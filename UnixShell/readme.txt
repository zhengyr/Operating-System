Your assignment is going to be to build a basic command line environment for interacting with a file system. This command line environment (or shell) will be written in java. The first part of this assignment is designed to test your coding ability in Java, and ensure that you have the coding pre-requisites to be able to complete this course. You will do this while learning about Unix and practicing your Java-based System calls.Essential to this assignment is an understanding the fundamentals of a UNIX shell, and the functionality that is expected from a UNIX like system.
